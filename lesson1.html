What is JavaScript?

JavaScript is a dynamic scripting language that allows developers to create multimedia web pages that are easily controlled and manipulated when used alongside HTML and CSS. Creator Brenden Eich took inspiration from Java, Self, and Scheme to create JavaScript in 1995. This was done because Netscape Communications, a company that created the Netscape Navigator browser, wanted to make their webpages more dynamic, with multimedia applications and activity.

Data Types and Variables

We just took a look at logging strings to the console. But what really is a string?

A string is one of many data types in JavaScript. Data types represent the types of data that can be stored as variables. There are two types of data types (LOL): primitive and non-primitive. Primitive types can only hold one singular value, while non-primitive types can hold several values.

For now, let's focus on primitive types. Primitive types include:

  • String: holds text in quotation marks or single quotes
  • Number: holds any number
  • BigInt: holds any number less than -2^63 and greater than 2^63-1
  • Boolean: logical; holds true and false values
  • Null: the variable has no value
  • Undefined: the variables has not yet been assigned a value
  • Symbol: represet and return unique identifiers

Variables can be defined using one of three keywords:

  • var: can be used in any situation; should only be used for older browsers
  • let: allows variables to be reassigned
  • const: cannot be reassigned after its instantiation

Wrap It Up!

To Wrap It Up!, JavaScript is a dynamic language created to make webpages dynamic and interactive. It usually goes hand-in-hand with HTML and CSS in order to design and add content to webpages. Code can be printed, or logged, to a panel called the console using the statement console.log(). Within the parantheses can be values of a variety of data types, including Strings, Numbers, BigInts, etc. Values of any of these data types can also be assigned to variables that will store them for later use. Arithmetic operators, such as + and -, can be used to perform mathematical operations on numerical values. The operator + can be used for concatenation.
