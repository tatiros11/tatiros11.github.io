Comparison and Logical Operators

Used commonly within conditional statements are comparison and logical operators. A comparison operator simply compares values. These comparisons then evaluate to a boolean value (true or false).

Comparison operators include:

  • >: greater than
  • <: less than
  • >=: greater than or equal to
  • <=: less than or equal to
  • ===: equal to
  • !==: not equal to

Wrap It Up!

To Wrap It Up!, conditional statements are statements that are used to execute certain code segments only when a specific condition is met. There are comparison operators that can be used within these statements, which will compare values and run the following code segment based on whether the comparison returned true or false. Finally, logical operators can be used to combine or negate conditions.
